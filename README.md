# ESAB — Ruffian EMP 270G Product Page  
### Features & Benefits Component (Vue 3)

---

## Overview

This project implements the **Features & Benefits section** for the ESAB Ruffian EMP 270G product page.
The goal was to build a **standalone, reusable Vue 3 component** that can be integrated into an existing CMS-driven product page.
The implementation follows a clean, component-based architecture and focuses on performance, responsiveness, and maintainability.
